# 閱後即毀 Vanish Mode
“「閱後即毀」功能使你能夠傳送敏感或暫時的訊息，它們稍後會在對話紀錄中被移除。”

"'Vanish Mode' or disappearing messages are helpful for sensitive or temporary information that you don't want to persist in the chat history."
